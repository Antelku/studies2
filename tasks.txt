Лабораторная работа №1, семестр 2
Задание 1. Создать дополнительную страницу, к основному проекту. Верстка должна быть выполнена на основе flex-контейнеров.

Задание 2. Создать "кнопки", слева от основного окна текущей страницы. придать закругленную форму (степень закругления не менее 5px). Использовать border, не менее 1px. Придать кнопкам фон, используя изображение (однотонный, фактурный узор). Применить эффект изменения яркости кнопки, без изменения яркости текста. Расположение кнопок - вертикальное с расстоянием равным между кнопками, но вдвое меньшим от верхнего и нижнего краев.

Задание 3. В основном окне создать не менее 4 контейнеров с контентом (изображение и текст). Расположение элементов свободное. Привязать фон каждой кнопки к фону соответствующего ей контейнера (пара - кнопка + контейнер). Применить к контейнерам тень, со степенью распространения не менее 5px. Расположение контейнеров - горизонтальное, с равным расстоянием между ними.

Задание 4. Создать контейнеры, без контента, но с применением разных фонов. Размер 200х200px. Размещение блока контейнеров - ниже элементов предыдущего задания. Размещение контейнеров внутри блока - по диагонали.


Лабораторная работа №2, семестр 2
Задание 1

Создать полноценную страницу используя flex-контейнеры, возможны два варианта:

1.      Продолжить текущую страницу, с сохранением условий предыдущего задания;

2.      Создать новую страницу.

Задание 2.

Создать Grid-контейнеры, с сеткой 4х3, используя в качестве размеров строк и столбцов, следующие элементы: fr, абсолютные величины и проценты. В итоге должны получиться три вида «сеток».

Задание 2.1.

Первая «сетка» должна быть выполнена с помощью функции repeat.

Задание 2.2.

Во второй «сетке» необходимо использовать функцию min-max.

Задание 2.3.

В третьей «сетке» необходимо разделить контейнеры с помощью «названия линий».

Использование горизонтального или вертикального размещения – по выбору.

 

Задание 3.

Написать пример «сетки», в которой элементы автоматически размесятся по ячейкам.

 

Задание 4.

Разместить элементы контента в первой «сетке» - во втором столбце, второй строке.

Во второй «сетке» - в четвертой строке, третьем столбце.

В третьей «сетке» - в первом столбце, первой строке.
